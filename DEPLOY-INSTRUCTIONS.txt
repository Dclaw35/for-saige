HOW TO UPDATE YOUR GITHUB PAGES SITE

1. Unzip this folder.
2. Open your `for-saige` repository on GitHub.
3. Click `Add file` > `Upload files`.
4. Drag in:
   - index.html
   - style.css
   - script.js
   - the full images folder
5. Commit changes.
6. If GitHub asks whether to replace files with the same name, allow it.
7. Wait 1 to 3 minutes.
8. Refresh your GitHub Pages link.

If GitHub Pages is not enabled yet:
Settings > Pages > Source: Deploy from branch > Branch: main > Folder: /root > Save.
