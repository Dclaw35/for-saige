HOW TO UPDATE YOUR GITHUB PAGES SITE

This is V4: the cinematic no-scroll slideshow.

1. Unzip this folder.
2. Open your `for-saige` repository on GitHub.
3. Delete or replace the old:
   - index.html
   - style.css
   - script.js
   - images folder
4. Upload everything from this folder:
   - index.html
   - style.css
   - script.js
   - images folder
5. Commit changes.
6. Wait 1 to 3 minutes.
7. Open your GitHub Pages link.

Important cache note:
This version uses style.css?v=4 and script.js?v=4. If you still see the old version, hard refresh:
Windows: Ctrl + F5
Mac: Command + Shift + R
iPhone: close the browser tab and reopen the link.
